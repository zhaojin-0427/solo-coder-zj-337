# 古礼席位摆放与仪程卡预演器 - 产品需求文档

## 1. 产品概述

古礼席位摆放与仪程卡预演器是一款面向汉服社、传统礼仪课程的纯前端可视化工具，帮助新手快速掌握揖礼、沃盥、献盏等古礼动作的站位顺序、器物摆放和行礼节拍。

- **目标用户**：汉服社成员、传统礼仪课师生、古礼爱好者
- **核心价值**：通过可视化拖拽和预演，降低古礼学习门槛，提升排练效率
- **使用场景**：礼仪课教学、社团排练、仪式彩排

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，本地使用 | 创建、编辑、保存方案，导出和打印 |

### 2.2 功能模块
1. **场景选择器**：迎宾礼、敬茶礼、冠礼展示、祭礼讲解四种预设场景
2. **席面画布**：俯视图画布，支持拖拽摆放席位、案几、器物
3. **器物素材库**：主位、宾位、案几、席垫、各种礼器卡片
4. **礼序步骤栏**：按礼序模板展示上场顺序、行礼方向、器物递送路线
5. **视图切换**：俯视图 / 流程图视图两种模式
6. **方案管理**：保存多套站位方案到 localStorage
7. **导出打印**：导出席位示意图、生成大字版仪程卡、打印流程卡

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主工作台 | 顶部导航栏 | 场景切换、视图切换、方案管理、导出打印按钮 |
| 主工作台 | 左侧素材库 | 器物分类、可拖拽的素材卡片列表 |
| 主工作台 | 中央画布区 | 席面画布/流程图，支持拖拽、缩放、选中编辑 |
| 主工作台 | 右侧步骤栏 | 礼序步骤列表，当前步骤高亮，步骤说明与方向指示 |
| 主工作台 | 底部状态栏 | 当前方案名称、元素数量、缩放比例 |
| 打印预览 | 仪程卡预览 | 大字版仪程卡，可分页打印 |

## 3. 核心流程

### 3.1 主使用流程
用户选择礼仪场景 → 从素材库拖拽元素到画布 → 调整位置和属性 → 参考礼序步骤栏调整站位 → 切换视图检查流程 → 保存方案 → 导出/打印

```mermaid
flowchart LR
    A["选择礼仪场景"] --> B["从素材库拖拽元素"]
    B --> C["在画布上调整位置"]
    C --> D["参考礼序步骤调整"]
    D --> E["切换视图检查"]
    E --> F["保存方案"]
    F --> G["导出图片/打印仪程卡"]
```

### 3.2 礼序预演流程
选择当前步骤 → 画布高亮相关元素 → 显示行礼方向箭头 → 展示器物递送路线动画 → 下一步

## 4. 用户界面设计

### 4.1 设计风格
- **设计主题**：古风雅致，仿宣纸质感
- **主色调**：赭石红 (#8B4513)、墨黑 (#2C2C2C)、米白 (#F5F0E6)
- **辅色调**：青玉色 (#5F9EA0)、鎏金色 (#D4AF37)
- **按钮风格**：圆角矩形，细微浮雕效果，悬停有轻微下沉
- **字体**：标题用楷体/宋体风格，正文用简洁易读的无衬线字体
- **布局风格**：三栏布局，左侧素材库、中央画布、右侧步骤栏
- **装饰元素**：祥云纹样、回纹边框、水墨晕染效果

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 主工作台 | 顶部导航栏 | 场景下拉选择、视图切换 tab、保存/导出按钮组 |
| 主工作台 | 左侧素材库 | 分类标签页、素材卡片网格、拖拽手势提示 |
| 主工作台 | 中央画布 | 宣纸质感背景、网格参考线、可拖拽元素、方向箭头 |
| 主工作台 | 右侧步骤栏 | 步骤时间轴、当前步骤卡片、动作说明文字 |
| 打印预览 | 仪程卡 | 大字标题、步骤列表、二维码占位、装饰边框 |

### 4.3 响应式
- 以桌面端为主要设计目标（1280px 以上）
- 平板端自适应三栏变两栏，步骤栏折叠
- 移动端单栏布局，素材库和步骤栏改为底部抽屉

### 4.4 动画与交互
- 元素拖拽：半透明跟随、放置时缩放回弹
- 步骤切换：淡入淡出过渡、高亮元素脉冲动画
- 礼序演示：箭头沿路径流动动画
- 悬停效果：细微阴影变化、边框颜色加深
